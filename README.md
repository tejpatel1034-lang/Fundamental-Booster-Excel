#  PR. 1 Fundamental Booster – Excel Project

##  Project Overview

This project is designed to strengthen fundamental Microsoft Excel skills through practical exercises and real-world datasets. It covers essential Excel functions, lookup formulas, text manipulation, date & time calculations, mathematical functions, and dynamic data analysis.

The workbook contains multiple worksheets where each task demonstrates the practical use of Excel formulas and features.

---

## Worksheets

- Students Grade
- Sales Data
- Employee Data

## Main Tasks

### Student Grade Analysis
- Classify grades using IF and Nested IF
- Count students using COUNTIFS
- Calculate average marks
- Filter top-performing students

### Sales Data Analysis
- Lookup product prices
- Calculate discounts
- Region-wise sales analysis
- Product-wise sales summary

### Employee Data Analysis
- Salary lookup using XLOOKUP
- Date of joining analysis
- Age calculation
- Employee information lookup using INDEX & MATCH
